IMU
This class represents the Inertial Movement Unit. It implements a singleton pattern, you don't need to create an instance of this class. You just need to call getInstance() to get a pointer to the unique instance of this class. The first call will create the unique instance. From this class you can acces to all the data recorded from all the sensors of the inertial unit.
