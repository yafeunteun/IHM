Detailed Description

This class represents a data storage object and provides storage for several sets of data each. It also furnishes methods to organize and to access the differents sets of data.

Constructor.

Constructor of DataHolder class

Parameters
list: A list of strings. Each string defines the label of a set of data. For instance let's say you want to store 3D points (i.e 3 vectors of data: x, y, z), the list could be {"x", "y", "z"}. If so, 3 sets will be created with the corresponding label.
type: The ProxyStrategy associated with the DataHolder, when data will be added to the instance, the data will be filtered before using the ProxyStrategy associated to the type chosen. See ProxyStrategy documentation for more.
